# 上线流程

## 提交代码后自动部署

```bash
git add .
git commit -m "改了什么"
git push
```

GitHub Actions 会自动部署到 https://safan-color-picker-mobile.pages.dev

## 首次配置

只需要在 GitHub 仓库设置里加一个 Secret：

1. 打开 https://github.com/Alienwang1980/Safan-color-picker-/settings/secrets/actions
2. 点 **New repository secret**
3. Name: `CLOUDFLARE_API_TOKEN`
4. Value: 存在 skill 引用文件里（cloudflare-pages-deploy）
5. 保存即可
