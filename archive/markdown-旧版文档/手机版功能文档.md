## 二、3D 场景与交互

### 模型结构（与 PC 版一致）
| Part 名称 | 可配色 | 材质 |
|---|---|---|
| Back_Color | ✅ | MeshStandardMaterial |
| Bar_Color | ✅ | MeshStandardMaterial |
| FrontCap_Color | ✅ | MeshStandardMaterial |

| Battery | ✅ | 特殊电池颜色 |
| Screw | ❌ | 固定银色金属质感 |

### 3D 渲染参数
- **相机**：PerspectiveCamera FOV=30，位置 (0,32,55)，lookAt (0,15,0)
- **渲染器**：WebGL，antialias=true，ACESFilmicToneMapping，SRGBColorSpace，pixelRatio=2
- **控制器**：OrbitControls，dampingFactor=0.08，minDistance=5，maxDistance=120，无 pan

### 模型 Y 轴偏移
- `mesh.position.y = 10`（补偿手机视角看到的模型位置）

### 灯光系统
| 灯光 | 类型 | 强度 | 位置 |
|---|---|---|---|
| 环境光 | AmbientLight | 0.55 | — |
| 主方向光 | DirectionalLight | 1.0 | (5,10,5) |
| 补光 | DirectionalLight | 0.3 | (-5,3,-5) |

## 三、配色系统

### 配色库（共用 PC 版数据结构）
```
COLOR_GROUPS:
  PLA Basic        → 33 色，type: "Basic"
  PLA Matte        → 25 色，type: "matte"
  PLA Translucent  → 10 色，type: "translucent"
```

### 特殊部件颜色
- **Battery**：使用 `window.__BATTERY_COLORS`（5 色：黑/白/粉/灰/橙）
- **Fan**：固定 2 色（灰色 #9E9E9E / 黑色 #212121）
- **Screw**：固定银色，metalness=0.95

### 材质渲染（applyColor）
手机版 `applyColor()` 直接设置 hex 颜色，**不区分材质类型**（无 roughness/metalness/emissive 差异），简化处理以适应弹窗快速应用场景。

## 四、UI 布局

### 全屏布局（从上到下）
```
┌─────────────────────────┐
│  Toolbar（固定顶部，50px） │
│  [Safan] [保存配色][随机]  │
├─────────────────────────┤
│                         │
│    3D Canvas（全屏）      │
│    点击部件 → 弹窗        │
│                         │
├─────────────────────────┤
│  Bottom Strip（可折叠）    │
│  [拖动条] [配色卡片列表]   │
└─────────────────────────┘
```

### Toolbar
- 左：标题 "Safan"
- 右：**保存配色**（btn-save）→ `saveCurrentProject()` + **随机**（btn-random）→ `randomizeColors()`

### 底部弹窗（Modal Bottom Sheet）
点击部件后弹出，全屏覆盖：
- Tab 栏：Basic / Matte / Translucent（点击切换）
- 色块网格：当前 Tab 下的所有颜色色块
- 关闭按钮

### 底部横条（Bottom Strip）
- 默认收起（collapsed），仅显示 90px 高度 + 拖动条
- 上滑或点击拖动条 → 展开显示配色卡片列表
- 配色卡片：显示预览色条，点击加载该方案的完整配色

## 五、保存与加载系统

### Project 数据结构
```javascript
{
  id: 1748553600000,
  name: "配色方案 1",
  colors: {
    Back_Color: "#C12E1F",
    Fan: "#9E9E9E",
    _previews: ["#C12E1F", "#FFFFFF", ...],  // 最多 5 个非灰色色值预览
  },
  savedAt: "2026-05-29T..."
}
```

> localStorage key: `sidan_projects_mobile`

### 操作
- **保存**：Toolbar「保存配色」按钮 → `saveCurrentProject()`
  - 自动读取所有部件当前颜色
  - 自动生成 `_previews` 预览色条（最多 5 个非默认灰色）
  - 自动命名「配色方案 N」
- **加载**：Bottom Strip 配色卡片点击 → `applyProjectColors(id)` + `expandDetailArea(id)`
  - 颜色渐变动效（350ms）
  - Detail 区展开显示该方案所有部件色块
  - 滚动卡片到视口中央
- **删除**：Detail 区部件 chip 点击 → `selectPart()` 重新打开弹窗修改

## 六、弹窗配色流程（核心交互）

```
点击3D模型部件
    ↓
openModal(partName)
    ↓
Tab 默认切到 "PLA Basic"（重置UI）
    ↓
renderModalGrid() → 显示 Basic 色块
    ↓
用户点击色块
    ↓
selectModalSwatch(index, colors)
    ├─ 保存 pendingColor
    ├─ applyColor(partName, hex)  → 实时预览
    ├─ animateColorChange()       → 渐变动画
    ├─ showToast("已应用: 颜色名")
    └─ closeModal()              → 自动关闭
```

### Tab 切换
- `initModalTabs()` 绑定 Tab 点击事件
- 切换 Tab → 更新 `modalColorType` → `renderModalGrid()` 重新渲染色块
- Tab 的 `data-type` 格式：`"PLA Basic"` / `"PLA Matte"` / `"PLA Translucent"`
- `getColorsByType()` 将 Tab type 映射为 `COLOR_GROUPS` key

## 七、动画系统

### 颜色渐变（animateColorChange）
- 持续时间：350ms
- 缓动：easeOutQuad
- 逐 Mesh 插值 color.r/g/b
- 运行中自动取消旧动画（`cancelAnimationFrame`）

### Bottom Strip 过渡动画
- CSS `transition: transform 0.3s ease`
- 展开/收起由 `.collapsed` class 控制

## 十、已知问题

1. **Tab type 映射**：`getColorsByType` typeMap 中 `'Basic'→'Basic'`、`'matte'→'Matte'`、`'translucent'→'Translucent'`，但 Tab `data-type` 是 `"PLA Basic"` 格式，存在映射不一致风险（实际工作正常因 HTML 中 Tab key 与 Map key 匹配）
2. **Battery 颜色未单独处理**：手机版 `applyColor` 直接调用 `_setMeshColor`，Battery 用的也是 `BATTERY_COLORS` 但在 `applyColor` 中未区分处理，与 PC 版一致
3. **透明部件渲染**：手机版 `applyColor` 无 `transparent=true` 处理，translucent 类型颜色无法真正实现半透明效果
4. **Screw 金属质感**：与其他可配色部件共用一套 `applyColor`，没有 PC 版 `_reapplyAllColors` 的完整材质重设逻辑
