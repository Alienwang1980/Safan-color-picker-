## 三、手机版需要新增的 UI 元素

### 3.1 配色模式切换（新增）

**位置：** Toolbar 右侧或 3D Canvas 左上角浮动按钮

**形态：** 两个轻量按钮 pill 形态
```
[ 原色 ] [ 参考模式 ]
```

**交互：**
- 点击切换 `colorMode`（`'official'` / `'reference'`）
- 高亮当前选中模式
- 切换后重新应用所有部件颜色（`_reapplyAllColors`）

**对应 PC 版代码：** `btn-original` + `btn-reference`（index_v2.html:385-386）

---

### 3.2 配色库删除入口（新增）

**位置：** Bottom Strip Detail 区或配色卡片长按

**形态：** 方案详情区下方增加删除按钮
```
[ 删除方案 ]（红色危险按钮）
```

**交互：**
- 点击 → `confirm('确定要删除...')` → `deleteCurrentProject(id)` → 重新渲染卡片列表

**对应 PC 版代码：** `btn-delete-project`（index_v2.html:407）

---

### 3.3 下载模型链接（可选新增）

**位置：** Toolbar 左上

**形态：** 文字链接或图标按钮

## 五、Bottom Strip 交互定义

### 5.1 收起状态（collapsed）
- 显示高度：90px
- 内容：拖动条 + 配色卡片横滑列表（最多可见几张）
- 拖动条视觉指示：居中灰色竖条，点击也可切换

### 5.2 展开状态（expanded）
- 高度：根据内容自适应或固定最大高度（约 50vh）
- 内容：拖动条 + 配色卡片列表 + Detail 区
- Detail 区：当前选中方案的部件 chip 列表
- 关闭方式：下滑超过 30px 或点击拖动条

### 5.3 配色卡片样式
- 尺寸：宽度固定（约 72px），高度约 54px
- 内容：`_previews` 色条（最多 5 格横排）或单色占位
- 选中态：卡片 `.center` class（更大/高亮边框）
- 非选中态：`.side`（缩小/透明）

## 六、底部弹窗（Modal Bottom Sheet）交互定义

### 6.1 打开动画
- 从底部滑入（transform: translateY(100%) → translateY(0)）
- 背景 overlay 淡入（opacity 0 → 0.5）
- 时长：300ms ease

### 6.2 Tab 栏
- 三个 Tab：Basic / Matte / Translucent
- 默认选中：PLA Basic
- 切换时：重渲染 color-grid

### 6.3 关闭方式
- 点击右上角"关闭"按钮
- 点击 overlay 遮罩区域
- （手机版不支持点空白处关闭，因为 canvas 全屏占位）
