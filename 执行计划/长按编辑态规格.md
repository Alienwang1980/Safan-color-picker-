# 配色库 — 长按编辑态（iOS Jiggle Mode）

> 规格来源：iOS 桌面长按进入编辑模式的交互规范
> 参考：Kirupa「Creating the iOS Icon Jiggle/Wobble Effect in CSS」+ Apple HIG 行为验证

---

## 交互规格

### 进入条件

| 条件 | 值 |
|---|---|
| 触发方式 | 长按卡片 500ms（手指不移动） |
| 判定 | `touchstart` 后 500ms 内 X/Y 位移 < 10px |
| 如果中途移动 | 清除定时器，正常滑动，不触发编辑态 |

### 进入行为（顺序）

1. **Haptic 震动反馈** — `navigator.vibrate(50)` 或 Haptic Feedback API
2. **所有卡片进入 wobble 动画** — 立即开始，无需等待
3. **删除按钮显现** — 每个卡片的左上角出现红色删除按钮
4. **停止当前动画的卡片** — 被长按的那张卡片暂停 wobble（突出选中态）

### 动画规格（Wobble Effect）

**CSS 动画参数：**

```css
/* 奇数卡片（2n-1） */
animation-name: keyframes-odd;
animation-duration: 0.28s;
animation-timing-function: ease-in-out;
transform-origin: 30% 5%;  /* 偏移中心点 */

/* 偶数卡片（2n） */
animation-name: keyframes-even;
animation-duration: 0.32s;
animation-timing-function: ease-in-out;
transform-origin: 50% 10%;  /* 偏移中心点 */

@keyframes keyframes-odd {
  0%   { transform: rotate(-1deg); }
  50%  { transform: rotate(1.5deg); }
  100% { transform: rotate(-1deg); }
}

@keyframes keyframes-even {
  0%   { transform: rotate(1deg); }
  50%  { transform: rotate(-1.5deg); }
  100% { transform: rotate(1deg); }
}
```

**动画特点：**
- 奇偶卡片交替旋转方向（-1deg ↔ +1.5deg），营造自然的整体晃动感
- `transform-origin` 偏移不在正中心，增强晃动感
- 每个卡片动画时长略有不同（0.28s / 0.32s），避免同步的机械感

### 删除按钮

**样式：**
- 位置：卡片左上角（`position: absolute; top: -6px; left: -6px`）
- 外观：红色圆形（24px × 24px），白色 "-" 符号
- 样式参考 iOS：`border-radius: 50%; background: #FF3B30; color: white;`

**交互：**
- 点击删除按钮 → 弹出确认提示 → 确认后删除方案 → 退出编辑态
- 删除按钮在编辑态期间始终显示

### 滑动行为

**关键规则：在编辑态下，横向滑动仍然有效**

- 用户可以继续左右滑动浏览配色库
- 所有卡片保持 wobble 动画
- 删除按钮保持可见
- 这是与 iOS Home Screen 一致的行为

### 退出编辑态

| 方式 | 触发条件 |
|---|---|
| 点击删除按钮并确认 | 删除该卡片，退出编辑态 |
| 点击卡片以外的区域 | 退出编辑态，停止 wobble |
| 切换到其他页面 | 退出编辑态 |

## 技术实现

### 状态管理

```javascript
// app_mobile_new.js
this.state.isEditMode = false;  // 编辑态标志

// touch 处理
let pressTimer = null;
let pressStartX = 0;
let pressStartY = 0;

card.addEventListener('touchstart', (e) => {
  pressStartX = e.touches[0].clientX;
  pressStartY = e.touches[0].clientY;
  
  pressTimer = setTimeout(() => {
    enterEditMode();  // 500ms 后进入编辑态
    navigator.vibrate(50);
  }, 500);
}, { passive: true });

card.addEventListener('touchmove', (e) => {
  const deltaX = Math.abs(e.touches[0].clientX - pressStartX);
  const deltaY = Math.abs(e.touches[0].clientY - pressStartY);
  
  if (deltaX > 10 || deltaY > 10) {
    clearTimeout(pressTimer);  // 用户在滑动，取消长按
  }
});

card.addEventListener('touchend', () => {
  clearTimeout(pressTimer);
});
```

### CSS 类控制

```css
/* 编辑态下所有卡片的 wobble 动画 */
.strip-card.jiggling {
  animation-name: keyframes-odd;  /* 或 keyframes-even */
  animation-duration: 0.28s;
  animation-timing-function: ease-in-out;
  animation-iteration-count: infinite;
}

/* 被长按的卡片暂停动画（保持静止以供操作） */
.strip-card.jiggling.active {
  animation-play-state: paused;
}
```

### 删除确认

```javascript
function confirmDelete(projectId) {
  // 显示确认弹窗（避免误删）
  if (confirm('确定删除这个配色方案？')) {
    deleteProject(projectId);
    exitEditMode();
  }
}
```

## 验证方式

1. 长按卡片 500ms → 所有卡片开始 wobble，左上角出现删除按钮
2. 进入编辑态后，横向滑动浏览 → 滑动正常，卡片保持 wobble
3. 点击删除按钮 → 弹出确认 → 确认后删除
4. 点击空白处 → 退出编辑态，wobble 停止
