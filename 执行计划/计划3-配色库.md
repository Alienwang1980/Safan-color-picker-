## 核心概念

### 焦点方案（Focused Project）

- **永远存在一个焦点方案**，所有颜色更改实时作用于它
- 改颜色 → 实时更新焦点方案的颜色数据
- 选卡片 → 加载该方案 + 成为新的焦点方案
- 点"添加配色方案" → 把当前模型状态快照存成新卡片（默认全灰）

### 相关术语

| 术语 | 说明 |
|------|------|
| 焦点方案 | 当前正在编辑的配色方案，`state.focusedProjectId` 标识 |
| 当前颜色 | 模型上各部件的实时颜色，所有更改实时同步到焦点方案 |
| 已存方案 | Strip 中已保存的方案卡片，点击加载到焦点 |
| 添加新方案 | 把当前模型状态（可能是全灰）快照为新卡片 |

## 步骤目标

**完成标志**：
1. 配色库 Strip 卡片横向排列，可左右滑动（容纳无限多）
2. 点"添加配色方案" → 把当前模型状态存为新卡片（焦点自动切换到新卡）
3. 点击非焦点卡片 → 加载颜色到模型 + 该卡成为焦点（卡片显示"显示详情"）
4. 点击焦点卡片 → 打开详情全屏页
5. 详情全屏页显示：大号缩略图 + 颜色列表 + 删除按钮 + 左上角返回
6. 详情页返回 → 回到编辑状态
7. 模型颜色实时同步到焦点方案（无需手动保存）

## 交互规格

### 配色库 Strip 卡片交互

| 操作 | 结果 |
|---|---|
| 点击**非焦点**卡片 | 加载颜色到模型 + 卡片成为焦点 + 卡片显示"显示详情" |
| 点击**焦点**卡片 | 打开详情全屏页 |
| 滑动卡片 | 正常滑动，无焦点变化 |
| 无已存方案 | Strip 为空，只显示"添加配色方案"按钮 |

### 焦点卡片视觉

- 焦点卡片：边框高亮（`.center`）+ "显示详情"文字覆盖在缩略图上
- 非焦点卡片：正常显示缩略图，无额外文字

### 详情全屏页（Detail Page）

- **进入**：点击焦点卡片
- **退出**：点击左上角返回按钮
- **内容**：
  1. 左上角返回按钮
  2. 大号缩略图（比 Strip 卡片大）
  3. 配色方案名称
  4. 颜色列表（色块 + 部件名 + 颜色名称 + 编号）
  5. 底部删除按钮

### 添加新配色方案

| 操作 | 结果 |
|---|---|
| 点击"添加配色方案"按钮 | 把当前模型状态存为新卡片，焦点切换到新卡 |

**新方案默认状态**：跟随当前模型颜色（首次打开项目时为全灰 `#888888`）

### 删除配色方案

- 点击详情页底部删除按钮 → 原生 `confirm()` 确认框
- 确认后删除 → 焦点方案切换到「最近添加的方案」或清空
- 删除后如果无方案，自动创建一个新的空方案（焦点状态）

### 修改 2 — 新增 createNewProject() 方法

**文件**：`app_mobile_new.js`

把当前模型状态存为新方案，设为焦点：
```javascript
createNewProject() {
  // 收集当前模型所有部件颜色
  const colors = {};
  // 模型未加载保护
  if (!this.state.parts || Object.keys(this.state.parts).length === 0) {
    return;
  }
  Object.entries(this.state.parts).forEach(([name, part]) => {
    if (part.mesh && part.mesh.material) {
      const c = part.mesh.material.color;
      colors[name] = '#' + c.getHexString();
    }
  });
  // 生成预览色（最多5个）
  const previewHexes = Object.entries(colors)
    .filter(([k, v]) => typeof v === 'string' && /^#[0-9A-Fa-f]{6}$/.test(v))
    .map(([, v]) => v)
    .slice(0, 5);
  colors._previews = previewHexes;

  const project = {
    id: Date.now(),
    name: '配色方案 ' + (this.state.projects.length + 1),
    colors,
    savedAt: new Date().toISOString(),
  };
  this.state.projects.push(project);
  this.state.focusedProjectId = project.id;
  this._saveProjects();
  this.renderBottomStripProjects();
}
```

### 修改 3 — 实时同步颜色到焦点方案

**文件**：`app_mobile_new.js` — `applyColor()` — 末尾新增实时同步逻辑

原有 `applyColor()` 保留（4行），在其后追加同步到焦点方案的逻辑：
```javascript
applyColor(partName, hex) {
  const part = this.state.parts[partName];
  if (!part) return;
  part.currentColor = hex;
  this._setMeshColor(part.mesh, hex);
  // --- 以下为新增：实时同步到焦点方案 ---
  const focused = this.state.projects.find(p => p.id === this.state.focusedProjectId);
  if (focused) {
    focused.colors[partName] = hex;
    // 更新预览色
    const previewHexes = Object.entries(focused.colors)
      .filter(([k, v]) => typeof v === 'string' && /^#[0-9A-Fa-f]{6}$/.test(v) && k !== '_previews')
      .map(([, v]) => v)
      .slice(0, 5);
    focused.colors._previews = previewHexes;
    this._saveProjects();
  }
}
```

```javascript
openDetailPage(projectId) {
  const project = this.state.projects.find(p => p.id === projectId);
  if (!project) return;

  // 生成大号缩略图 canvas（reuse renderer）
  const prevTarget = this.renderer.domElement;
  this.renderer.domElement = this.detailCanvas;
  this.renderer.render(this.scene, this.camera);
  const thumbnail = this.detailCanvas.toDataURL('image/png');
  this.renderer.domElement = prevTarget;

  const partNames = Object.keys(project.colors || {}).filter(k => k !== '_previews');

  const html = `
    <div id="detail-page">
      <div class="detail-page-header">
        <button id="btn-detail-back">← 返回</button>
      </div>
      <div class="detail-page-thumb">
        <img src="${thumbnail}" alt="缩略图" />
      </div>
      <div class="detail-page-title">${project.name}</div>
      <div class="detail-page-list">
        <div class="detail-list-header">
          <span></span><span>部件</span><span>颜色名称</span><span>编号</span>
        </div>
        ${partNames.map(name => {
          const hex = project.colors[name];
          const colorObj = this.findColorByHex(hex);
          return `
            <div class="detail-list-row">
              <span class="detail-swatch" style="background:${hex}"></span>
              <span>${this.formatName(name)}</span>
              <span>${colorObj ? colorObj.name : '—'}</span>
              <span>${colorObj ? (colorObj.code || '—') : '—'}</span>
            </div>
          `;
        }).join('')}
      </div>
      <div class="detail-page-footer">
        <button id="btn-detail-delete" data-id="${projectId}">删除配色方案</button>
      </div>
    </div>
  `;
  this.detailContainer.innerHTML = html;
  this.detailContainer.classList.add('open');

  // 绑定返回按钮
  document.getElementById('btn-detail-back').addEventListener('click', () => {
    this.closeDetailPage();
  });
  // 绑定删除按钮（confirm 在 deleteProject 内部统一处理）
  document.getElementById('btn-detail-delete').addEventListener('click', (e) => {
    const pid = parseInt(e.target.dataset.id);
    this.deleteProject(pid);
    this.closeDetailPage();
  });
}

closeDetailPage() {
  this.detailContainer.classList.remove('open');
  this.detailContainer.innerHTML = '';
}
```

### 修改 9 — 删除方案逻辑

**文件**：`app_mobile_new.js` — `deleteProject(id)`

```javascript
deleteProject(id) {
  if (!id) return;
  const project = this.state.projects.find(p => p.id === id);
  if (!project) return;
  if (!confirm('确定要删除这个配色方案吗？')) return;
  this.state.projects = this.state.projects.filter(p => p.id !== id);
  this._saveProjects();
  // 如果删除的是焦点方案，切换到最近一个或创建新空方案
  if (this.state.focusedProjectId === id) {
    if (this.state.projects.length > 0) {
      const latest = this.state.projects[this.state.projects.length - 1];
      this.state.focusedProjectId = latest.id;
      this.applyProjectColors(latest.id);  // 末尾已调 renderBottomStripProjects，无需再调
    } else {
      // 无方案时创建新的空方案
      this.createNewProject();  // 末尾已调 renderBottomStripProjects
      return;  // 已重建，跳过末尾的 renderBottomStripProjects
    }
  }
  this.renderBottomStripProjects();  // 非焦点删除时才需要重建
}
```

### 修改 11 — renderBottomStripProjects 结构更新 + flash 动画

**文件**：`app_mobile_new.js` — `renderBottomStripProjects()`

重写整个方法，核心变更：
1. 空方案时也要显示 `+` 按钮（不提前 return）
2. 卡片 HTML 中用 `focusedProjectId` 判断 isCenter
3. 末尾统一处理焦点卡片的 flash 动画

```javascript
renderBottomStripProjects() {
  const container = document.getElementById('strip-cards');
  if (!container) return;

  const addBtn = `<button class="btn-add-project" id="btn-add-project">+</button>`;
  let cardsHtml = '';

  if (this.state.projects.length > 0) {
    cardsHtml = this.state.projects.map((p, i) => {
      const previewColors = (p.colors || {})._previews || [];
      const colorBars = previewColors.length > 0
        ? previewColors.map(hex => `<span style="background:${hex};flex:1;"></span>`).join('')
        : '<span style="flex:1;background:#e0e0e0;"></span>';
      const isFocused = this.state.focusedProjectId !== null && p.id === this.state.focusedProjectId;
      const detailLabel = isFocused
        ? '<div class="strip-card-detail-label">显示详情</div>'
        : '';
      return `
        <div class="strip-card${isFocused ? ' center' : ' side'}" data-project-id="${p.id}" data-index="${i}">
          <span class="delete-btn" data-delete-id="${p.id}">×</span>
          <div class="strip-card-colors">${colorBars}</div>
          ${detailLabel}
        </div>
      `;
    }).join('');
  }

  container.innerHTML = addBtn + cardsHtml;

  // 绑定卡片点击（编辑模式保护已在修改4中处理）
  container.querySelectorAll('.strip-card').forEach(card => {
    card.addEventListener('click', () => {
      if (this.state.isEditMode) return;
      const pid = parseInt(card.dataset.projectId);
      if (pid === this.state.focusedProjectId) {
        this.openDetailPage(pid);
      } else {
        this.state.focusedProjectId = pid;
        this.applyProjectColors(pid);
      }
    });
  });

  // 末尾统一处理 flash 动画
  const focusedCard = container.querySelector(
    `.strip-card[data-project-id="${this.state.focusedProjectId}"]`
  );
  if (focusedCard) {
    focusedCard.classList.add('flash');
    setTimeout(() => focusedCard.classList.remove('flash'), 500);
  }
}
```
